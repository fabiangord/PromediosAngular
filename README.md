
# App promedios

Aplicacion en la que puedes ingresar notas y te dará el promedio de estas notas.

Diseñada con Angular.

## Servidor de desarrollo

Ejecuta 'ng serve -o' y se abrira la ventana, recuerda tener instalado Angular
